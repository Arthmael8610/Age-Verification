==========
How To Use
==========
Place ageVerification in your web directory
include the file in your page with require_once 'ageVerification.php';
initiate the class with
    $destictive_name = new ageVerification();
    $destictive_name -> getAge($month, $day, $year);
    $month, $day, $year being your users input.

==========
About
==========

Ever wanted to require your users to be a certain age before entering your site, well not you can with this simple
script.
